# Phase 8: Jurisdictions Tab Polish — Plan

## Goal
Polish the Jurisdictions tab with a global toast notification system and integrate it into copy-to-clipboard actions on JurisdictionCard.

## Pre-existing Features (Phase 5)
- Copy-to-clipboard with 2-second visual feedback (Check icon)
- Password show/hide toggle
- External link for portal URL
- Phone as tel: link

## Tasks

### T1: Verify pre-loaded jurisdiction data
Start server and confirm all 7 jurisdictions are seeded and display correctly.

### T2: Add Toast notification system
Create `client/src/components/Toast.jsx` with:
- Context-based ToastProvider
- `useToast()` hook returning `addToast(message, type, duration)`
- Fixed top-center overlay, pill-shaped toasts
- Auto-dismiss after 2500ms
- Manual dismiss (X button)
- Success (green) and error (red) variants
- Wrap App with ToastProvider in main.jsx

### T3: Update JurisdictionCard to use toast on copy
- Import useToast in JurisdictionCard
- Add `label` prop to CopyButton
- Call `toast?.(label + ' copied!')` on clipboard write
- Pass `label="Username"` and `label="Password"` to respective CopyButtons

### T4: Build verify
Run `npx vite build` and fix any issues.

## Files Modified
- `client/src/components/Toast.jsx` (new)
- `client/src/main.jsx` (wrap with ToastProvider)
- `client/src/components/JurisdictionCard.jsx` (useToast in CopyButton)
