#### Components:
1. **Date range picker** — 4 preset buttons (This Week, This Month, This Year, All Time) + custom date inputs with Apply button
2. **Summary cards** — 3-column grid: Total / Active / Completed counts
3. **Pipeline Funnel** — horizontal BarChart (layout="vertical") from Recharts, one bar per stage, colored by stage color
4. **Stage Conversion Rates** — table rows per adjacent-stage pair, highlights worst conversion in red
5. **Activity Over Time** — stacked/grouped BarChart showing new_leads and completed per week

#### Key implementation details:
- `STAGE_COLORS` map: color name → hex (slate, blue, violet, green, yellow, orange, teal, pink, amber)
- Bar Cell fill uses `STAGE_COLORS[entry.stage.color]` — NOT Tailwind CSS variable syntax
- Custom Tooltip component for Recharts (dark theme: bg-slate-800)
- `useQuery` from @tanstack/react-query with queryKey `['analytics', from, to]`
- Fetch: `GET /api/analytics?from=...&to=...`
- `worstConversion`: sorted by rate ascending, highlighted with red bg + ⚠ icon
